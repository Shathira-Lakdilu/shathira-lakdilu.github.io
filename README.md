# Shathira Lakdilu - PCB Design Engineer Portfolio

Professional portfolio website showcasing PCB design projects for medical devices, wearables, and embedded systems.

## ЁЯЪА Quick Start - Deploy to GitHub Pages

### Step 1: Upload Files to GitHub

1. Go to your repository: `https://github.com/shathira-lakdilu/shathira-lakdilu.github.io`
2. Click "Add file" тЖТ "Upload files"
3. Drag and drop these files:
   - `index.html`
   - `style.css`
   - `README.md`
4. Click "Commit changes"

### Step 2: Enable GitHub Pages

1. Go to repository "Settings"
2. Click "Pages" in left sidebar
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select "main" and "/ (root)"
5. Click "Save"

### Step 3: Wait 2-3 Minutes

Your site will be live at: `https://shathira-lakdilu.github.io`

## ЁЯУ╕ Adding Your PCB Images

1. In your repository, click "Add file" тЖТ "Upload files"
2. Create a folder called `images` (just type `images/` before filename)
3. Upload your blurred PCB images with these names:
   - `fetal-monitor.jpg`
   - `ecg-garment.jpg`
   - `heart-simulator.jpg`
   - `imu-module.jpg`
   - `am62-som.jpg`
   - `imx-som.jpg`
   - `usb-pd.jpg`
   - `bms.jpg`
   - `motor-driver.jpg`
   - `led-driver.jpg`
   - `antenna.jpg`

4. Then edit `index.html` and replace:
```html
<div class="placeholder">PCB Image<br>(Add your image)</div>
```
with:
```html
<img src="images/fetal-monitor.jpg" alt="Fetal Heart Monitor PCB" style="width: 100%; height: 100%; object-fit: cover;">
```
